# Task 1:-Construct an object using Parametric Design Technique

* Object:- Laptop Stand Prototype.

* Time needed:- less than 1min.

Procedure:-

    1. For design purposes I am using Fusion 360 Software.

    2. To start with I created a base sketch as shown in the picture on the left side. Then using the sketch tool I set forth the dimensions(225mm * 125mm,22degrees) . You can also set the per-defined parameters in the parameter section, so you can make changes when you insert your design in Laser CAD software.

    3. For the Rectangle board dimensions of (230mm * 130 mm) with four small rectangles of (20mm ** 10 mm) are inserted . The four rectangles will act as a hole to fit the two base /legs of the stand into the board . With this our design is completed.

    4. For cutting the desired design , we need to command Laser Cutter regarding material , power to be used , oringin and dimensions.

    5. For this I used Laser CAD v8.17 software. Here after importing the design from fusion 360 , we need to set material to be used(cardboard) ,origin(X,Y,Z axis), width of material and power to used according to material .

    6. After this send command to Laser cutter machine . After completing the origin setting in machine start the cutting process. In this way we obtained a precise and accurate cutting of design. Make sure to measure it again with mechnical device 'vernier caliper' to test the accuracy of machine.

    # Task 2:-Engraving using Bitmaping technique.

    * Object:-Engraved Mdf Photo Frame.

    * Time needed:- approx 1hr.

    Procedure:-

      1. Here First I assembled the components for Photo Frame using 'Canva' a design tool software used for editing , creating posters etc.

      2. Then I imported assembled frame in a photoshop software 'ImagR' which is widely used for setting the photo needed for engraving. Here you can crop your unnecessary elements form photo and then resize it as well.

      3. I used some special tools to add effect to my frame .That were:- Gamma Adjustment(to colour correct my frame),Sharpening(to make photo more detailed),material algorithm(to select proper wood material) .

      4. Download the finish product in png. extension. And now you can export it to Laser CAD for further process.

      5. After importing it to Laser CAD ,set the dimensions, origin, material width and power needed in the software. Give the command to your cutter machine and within 1hr of time span you get your desire product.

      In this way you can use your creativity in combine with tech tools to create amazing products.
