HTML (Hypertext Markup Language) invented in 1989 and CSS (Cascading Style Sheets) originated in 1996 are scripting languages used to create a web page and web applications. HTML provides web page structure,whereas CSS is mainly used to control web page styling.

Here we have used basic HTML and CSS to create a protfolio website. In this web-dev module conducted for 1 week we were introduced to what and how html and css are used to create the web pages. Implementing the learnings of week , I have created a protfolio website where i added different sections through which you get to know more about me, and my progress as i explore new technologies and can reach out to me as well!

Using css i have tried to add different effects like flex box, hover option, icons, clicable links ,sliders ,etc. intead of creating different css sheets , i incorporated it in html itself by adding it below title tag. Tags,lables and sign plays an important role in css.Different types of tags are anchor tag,colour tag, size tag ,spinning tags etc. CSS mainly function on its properties.

Main properties that i have used in my website are

align-text :-to align text,

animation:- to display animation,

background:-to display/attach background image,colour.position,size,etc,

flex:-to diplay colour blurring property,

hover:-to pop up the specific button,

img src:-to attach images,

a href:-to attach clicable links, and many more.
